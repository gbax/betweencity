Реализована программа для сокращения затрат ресурсов, таких как денежные средства, время, топливо, оборудования,
а также для предоставления возможности учитывать предпочтения условий для пользователя
(например, качество дорог, ограничения скорости на дорогах и т.д.) с использованием следующих методов:
• метод ветвей и границ;
• генетический алгоритм
• метод замещений и предпочтений Кини-Райфа.
В верхней части окна представлено меню, которое содержит следующие кнопки:
• Начать сначала;
• Назад;
• Далее;
• Справка;
• Об авторе.
При нажатии на кнопку «Начать сначала», можно перейти на первую страницу.
При нажатии на кнопку «Назад», можно вернуться на предыдущую страницу.
При нажатии на кнопку «Далее» можно перейти на следующую страницу.
При нажатии на кнопку «Справка» можно увидеть инструкцию по использованию программы.
При нажатии на кнопку «Об авторе» можно увидеть информацию о создателе программы.
На странице для ввода городов необходимо ввести название города, затем нажать на кнопку "Добавить".
Введенные города появятся в левой части экрана.
Также можно удалять города из списка нажав на кнопку "-" у соответствующего города.
На странице для ввода значений для построения одномерных функций ценности необходимо
ввести значения для построения одномерных функций ценности для критериев «Среднее ограничение скорости»,
«Среднее качество дорог», «Среднее количество светофоров на 1км»: наибольшее значение,
среднее значение по ценности 0.75, среднее значение по ценности 0.5, среднее значение по ценности 0.25
и наименьшее значение. Значения для построения одномерных функций ценности должны вводиться в порядке
убывания от наибольшего значения до наименьшего. На странице для ввода значений для определения шкалирующих коэффициентов необходимо ввести значения для
определения шкалирующих коэффициентов для критериев «Среднее ограничение скорости», «Среднее качество дорог»,
«Среднее количество светофоров на 1 км»: рейтинг критерия, значение для сравнения весов критериев. Рейтинг
критериев для определения шкалирующих коэффициентов для критериев «Среднее ограничение скорости», «Среднее качество дорог»,
«Среднее количество светофоров на 1 км» должен быть верно проранжирован. Значения для сравнения весов критериев для
определения шкалирующих коэффициентов для критериев «Среднее ограничение скорости», «Среднее качество дорог»,
«Среднее количество светофоров на 1 км» должны быть не больше наибольшего значения соответствующего критерия и не меньше
наименьшего значения соответствующего критерия.
На странице для ввода оценок средних разрешенных скоростей необходимо ввести оценки средних разрешенных скоростей
для каждой пары городов, которые должны быть не больше наибольшего значения соответствующего критерия и не меньше
наименьшего значения соответствующего критерия.
На странице для ввода оценок среднего качества дорог необходимо ввести оценки среднего качества дорог
для каждой пары городов, которые должны быть не больше наибольшего значения соответствующего критерия и не меньше
наименьшего значения соответствующего критерия.
На странице для ввода оценок среднего количества светофоров на 1 км необходимо ввести оценки среднего количества светофоров
на 1 км для каждой пары городов, которые должны быть не больше наибольшего значения соответствующего критерия и не меньше
наименьшего значения соответствующего критерия.
На странице для настройки генетического алгоритма пользователь имеет возможность настроить генетический алгоритм. Пользователь может поставить флажок
на против «Использовать мутацию», и в генетическом алгоритме будет использоваться мутация. В противном случае генетический
алгоритм будет работать без использования мутации. Также пользователь может выбрать число итераций в генетическом алгоритме,
что может повлиять на результат работы алгоритма и на скорость его работы.
На странице с результатами пользователь может наблюдать маршруты, изображенные на картах, вычисленные следующими методами:
• Метод ветвей и границ;
• Генетический алгоритм;
• Метод ветвей и границ с использованием метода замещений и предпочтений Кини-Райфа;
• Генетический алгоритм с использованием метода замещений и предпочтений Кини-Райфа.
Помимо маршрутов на карте изображены соответствующие пути, стоимости этих путей, а также коэффициенты стоимости путей по
модификационной матрице методом замещений и предпочтений Кини-Райфа.
